# paperclip-ceo-brief

> Structured briefing template for kicking off an AI CEO agent session.

---

## Description

A template for CEO Session Briefs — the structured briefings sent to an AI CEO agent at the start of a Paperclip session. The CEO reads all six business files, then produces a proposal (org structure, task breakdown, hiring request) for board review.

Use this template for every CEO session, first or returning.

---

## When to Use

Trigger on:
- "start a CEO session"
- "brief the CEO"
- "kick off a Paperclip session"
- Any time you need to hand context to an AI CEO agent before it begins work

---

## Install

Copy this folder into your opencode skills path:

```bash
cp -r paperclip-ceo-brief /path/to/your/skills/
```

Then ensure your `opencode.json` points to that path:

```json
{
  "skills": {
    "paths": ["/path/to/your/skills"]
  }
}
```
