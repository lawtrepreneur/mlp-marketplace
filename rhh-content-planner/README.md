# rhh-content-planner

> Plan, draft, and assess Substack posts for the Clarity > Complexity publication.

---

## Description

A content skill for the *Clarity > Complexity* Substack (romeshh.com), built on the BRAWSA methodology. Covers the full content workflow: pillar assignment, pre-draft intake, format selection, draft generation, and draft assessment.

Supports two post formats:
- **Short** (~400 words) — high-density, direct
- **Long** (~1,000+ words) — deeper exploration with structured sections

Three content pillars:
- **The Stakes** — why this matters now
- **In Practice** — how to apply it
- **Mental & Digital Models** — frameworks for thinking

Two reader avatars: CREDPRO (credentialed professionals) and EXEC-DEV (executive development).

Includes reference files: `brawsa-prompt.md` and `post-template.md`.

---

## When to Use

Trigger on:
- "write a post", "draft this", "assess this draft"
- "which pillar does this belong to"
- "content plan", "plan a post"
- Any Clarity > Complexity content task — always load this skill before drafting

---

## Install

Copy this folder into your opencode skills path:

```bash
cp -r rhh-content-planner /path/to/your/skills/
```

Then ensure your `opencode.json` points to that path:

```json
{
  "skills": {
    "paths": ["/path/to/your/skills"]
  }
}
```
